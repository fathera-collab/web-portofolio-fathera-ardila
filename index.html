Tentang Saya

Salam kenal! Saya Fathera Ardila, biasa disapa Fathera. Lahir di Lamongan pada 18 Maret 2006, saat ini saya sedang menempuh pendidikan S1 di Universitas Negeri Surabaya. Fokus utama saya ada pada eksplorasi web development dan desain UI/UX. Di waktu luang, saya biasanya menyalurkan energi dengan berenang atau mengunjungi spot-spot aesthetic untuk sekadar berfoto. Saya adalah pribadi yang senang belajar, cepat beradaptasi, dan selalu berkomitmen penuh untuk berkolaborasi secara maksimal di mana pun saya berada.

Dompetku

DompetKu adalah manajemen keuangan pribadi berbasis mobile yang dirancang untuk membantu pengguna mengelola transaksi, memantau tagihan, dan melihat laporan finansial secara praktis dalam satu genggaman. Dengan antarmuka yang bersih dan modern, aplikasi ini memudahkan siapa saja untuk mencatat pengeluaran dan pemasukan harian demi mencapai kestabilan finansial.

Golite

GoLite Reservation System adalah aplikasi desktop berbasis Java yang dirancang untuk mempermudah pemesanan layanan transportasi secara efisien. Aplikasi ini memungkinkan pengguna mendaftar sebagai penumpang atau driver, melakukan pemesanan, serta menerima bukti transaksi (faktur) secara otomatis.
