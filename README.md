## 🚀 Fitur Utama
* **Responsif:** Tampilan menyesuaikan secara otomatis untuk layar Desktop, Tablet, dan Mobile.
* **Modern & Clean UI:** Desain antarmuka yang bersih, elegan, dan mudah dinavigasi.
* **Interaktif:** Dilengkapi dengan animasi transisi yang halus dan fitur pemutar musik (*music player*) mini terintegrasi untuk pengalaman pengguna yang unik.
* **Cepat & Ringan:** Dibangun menggunakan teknologi web murni tanpa *framework* yang berat.

## 🛠️ Teknologi yang Digunakan
* **HTML5** (Struktur Semantic)
* **CSS3** (Styling, Flexbox, Grid, Animasi)
* **Vanilla JavaScript** (Interaktivitas dan DOM Manipulation)
